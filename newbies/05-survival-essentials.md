# 生存必備知識 (Survival Essentials)

## 資金安全 (Money Safety)
- 避免隨身攜帶過多現金以防強盜 (Muggers)
- 安全存放管道：
  - 派系銀行 (Faction Bank)：需確認派系規則
  - 點數市場 (Point Market)：現金換成點數 (Points)
  - 道具存儲：購買容易脫手的物資（如 Xanax）保值

## 毒癮與戒毒 (Addiction & Rehab)
- 藥物副作用：使用 Xanax 或 Ecstasy 會累積毒癮，降低戰鬥屬性
- 戒毒時機：屬性削弱顯著時，飛往瑞士 (Switzerland) 進行戒毒 (Rehab)
- 成本平衡：合理安排飛行時間與預算

## 基礎術語 (Terminology)

| 術語 | 說明 |
| :--- | :--- |
| Nerve (N) | 神經值，用於犯罪行為 |
| Energy (E) | 體力值，用於攻擊、健身房訓練 |
| Happiness (H) | 快樂值，影響訓練效率 |
| Xanax | 每次增加 250E 的體力藥物 |
| Donator (DP) | 贊助商狀態包，提升上限與恢復速度 |

## 每日習慣
- 轉盤 (Wheels)：每天至賭場 (Casino) 轉幸運轉盤
- 城市物品：每天檢查城市地圖拾取掉落物
