# Torn 術語表 (Terminology)

本表列出在筆記中應保留使用英文的關鍵字詞及其對應的中文說明。

## 基礎屬性與資源 (Basic Stats & Resources)

| 術語 | 說明 |
| --- | --- |
| Energy | 能量，用於訓練、攻擊、打工、犯罪 2.0 等。 |
| Nerve | 神經值，用於進行犯罪行為。 |
| Merits | 功勳，透過達成成就獲得，可用於升級被動技能。 |
| Battlestats | 戰鬥屬性，包含以下四項： |
| Strength | 力量，影響攻擊造成的傷害量。 |
| Defense | 防禦，減少受到的傷害量。 |
| Speed | 速度，影響攻擊的命中率。 |
| Dexterity | 靈巧，影響閃避敵方攻擊的機率。 |
| Happy | 快樂值，影響健身房訓練與部分工作的增益效果。 |
| Life | 生命值，戰鬥中的血量。 |

## 工作相關 (Work Related)

| 術語 | 說明 |
| --- | --- |
| Workstats | 工作屬性，包含以下三項： |
| Manual | 體力屬性。 |
| Intelligence | 智力屬性。 |
| Endurance | 耐力屬性。 |
| Job Points (JP) | 工作點數，每日上班獲得，可用於兌換公司福利。 |
| Director | 公司負責人。 |
| Trains | 公司負責人發放給員工的屬性訓練加成。 |

## 物品與冷卻時間 (Items & Cooldowns)

| 術語 | 說明 |
| --- | --- |
| Booster Cooldown | 增益冷卻，使用 Candy, Cans (能量罐), Alcohol 後增加。 |
| Drug Cooldown | 藥物冷卻，服用藥物如 Xanax 後增加。 |
| Medical Cooldown | 醫療冷卻，使用血袋或醫療用品後增加。 |
| Xanax | 贊安諾，主要能量來源藥物 (提供 250 能量)。 |
| SE (Stat Enhancer) | 屬性增強劑，昂貴消耗品，直接提升 1% 的單項戰鬥屬性。 |
| SE Cap | 財務效益門檻，指使用 SE 比能量罐更划算的屬性臨界點。 |
| Cans / Energy Drinks | 能量罐，增加能量並佔用 Booster Cooldown。 |

## 犯罪與戰鬥機制 (Crime & Combat)

| 術語 | 說明 |
| --- | --- |
| CE (Crime Experience) | 犯罪經驗，決定 NNB 高低的隱藏數值。 |
| NNB (Natural Nerve Bar) | 自然神經條，基礎的最大神經值上限。 |
| OC (Organized Crime) | 組織犯罪，由派系成員合作進行的大型犯罪。 |
| Faction | 派系 / 幫派。 |
| Warring | 戰爭，包含 Ranked War (RW) 排名戰。 |
| Attacking | 攻擊其他玩家。 |
| Chaining | 連鎖，派系藉由連續攻擊特定名單來獲取獎勵。 |
| Stealth | 隱密性，決定在攻擊後是否會暴露身份。 |
| Critical Hit | 暴擊。 |

## 遊戲行為與術語 (General Gameplay)

| 術語 | 說明 |
| --- | --- |
| Happy Jump | 快樂跳，利用補滿快樂值後消耗大量能量以獲取極高訓練收益的戰法。 |
| PI (Private Island) | 私人島嶼，目前遊戲中最高級的房產。 |
| Travel / Flying | 出國旅遊，主要目的是購買物品帶回 Torn 賺取價差。 |
| Rehab | 復健，前往瑞士降低藥物成癮度。 |
| Addiction | 藥物成癮，會降低工作屬性與員工效率。 |
| Points | 點數，特殊貨幣，可用於兌換多種功能。 |
| Level Holding | 壓級，獲得足夠經驗提升等級卻不點選升級按鈕。 |
| Networth (NW) | 使用者的總淨資產估計。 |

## 數值單位 (Number Units)

| 術語 | 說明 |
| --- | --- |
| k | 1,000 (千)。例如 10k = 10,000 (1 萬)。 |
| m | 1,000,000 (百萬)。例如 100m = 1 億。 |
| b | 1,000,000,000 (十億)。例如 1b = 10 億。 |
